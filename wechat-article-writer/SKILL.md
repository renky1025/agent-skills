# 自媒体文章写作

用专业的新媒体写作方法论，创作有流量、能传播的自媒体文章。

## 核心原则

- 选题定生死：选题决定流量天花板
- 标题决胜负：1秒内决定用户是否点击
- 结构抓人心：让读者读完并转发
- 情绪促传播：替读者说出他们想说的话

## 第一步：选题定生死

### 1.1 两大爆款选题来源

**追热点**（通往10w+的最短路径）

要快、准、狠。4个追热点套路：
- 改编热点句式：用热点事件的句式改编新内容
- 快速梳理事件：第一时间整理事件来龙去脉
- 复制高赞评论：直接使用热点下的神评论作为切入点
- 切入核心争议点：找出事件中引发讨论的关键分歧

**挖共识**（瞄准时代永恒痛点）

可提前储备。使用"爆款十大元素"公式：

将热点与以下元素组合：
- 三大感情：爱情、友情、亲情
- 五种情绪：愤怒、怀旧、暖心、愧疚、孤独
- 两大因素：地域、群体

示例：《躲在星巴克里的中年人：假装上班，是成年人最后的体面》
（热点"失业" + 群体"中年" + 地域"星巴克"）

### 1.2 高级心法

- 换角色：当所有人都在写A角度时，你去写B角度
- 硬组合：把两个不相关的东西结合，制造认知冲击
- 小见大：用一个小细节、小故事映射宏大主题

## 第二步：标题决胜负

### 2.1 核心心法

标题必须在1秒内回答读者："这与我有关吗？" "这对我有用吗？"

### 2.2 九大爆款模板

| 模板类型 | 示例 | 适用场景 |
|---------|------|---------|
| 数字法 | 95%的女人都爱上过渣男 | 数据支撑的观点 |
| 对比法 | 月薪3000的快乐，月入3万的人想象不到 | 反差制造冲突 |
| 疑问法 | 为什么有人喜欢渣男？ | 反常识观点 |
| 好奇法 | 微信这个新功能，检验一个人爱不爱你 | 话只说一半 |
| 高赞评论法 | [直接使用热点下的神评论] | 热点借势 |
| 身份标签法 | 35岁中年人，正在被职场抛弃 | 群体共鸣 |
| 利益承诺法 | 学会这3招，让你少走10年弯路 | 实用价值 |
| 情绪冲击法 | 看完这篇文章，我哭了整整一夜 | 情感共鸣 |
| 悬念揭秘法 | 那个辞职回老家的人，后来怎么样了 | 故事续集 |

### 2.3 标题生产流程

1. 为每个选题生成10-15个标题
2. AI筛选出5个最佳标题
3. 根据传播力、点击率、情感强度排序
4. 给出推荐理由

## 第三步：结构抓人心

### 3.1 黄金开头（3秒定生死）

开头要像钩子一样抓住读者：

- 冲突先行：直接抛出最激烈的矛盾点
- 悬念开场：留下问题不立即解答
- 数据冲击：用惊人数字震撼读者
- 场景代入：让读者瞬间身临其境
- 金句破题：用一句戳心的话开场

### 3.2 故事模板

**4步框架**：
1. 一句话概括核心故事
2. 定冲突（核心矛盾是什么）
3. 给理由（为什么会这样）
4. 来组合（把要素组装起来）

**3步完善**：
1. 交代背景（时间、地点、人物）
2. 补充核心信息（关键细节）
3. 触发情绪爆点（传播的本质）

**2招引爆**：
- 打乱叙事顺序（倒叙、插叙）
- 进行场景演绎（画面感描写）

### 3.3 四大文章框架

根据文章类型选择：

**框架1：观点 + N个事例**
- 适合：观点类文章、热点评论
- 结构：核心观点 → 事例1 → 事例2 → 事例3 → 总结升华

**框架2：大观点 + N个小观点**
- 适合：反常识观点、深度分析
- 结构：核心观点 → 分论点1 → 分论点2 → 分论点3 → 结论

**框架3：观点 + N个角度**
- 适合：素材丰富的综合性文章
- 结构：核心观点 → 角度1 → 角度2 → 角度3 → 总结

**框架4：观点 + 人物的N个故事**
- 适合：人物稿、纪实故事
- 结构：人物印象 → 故事1 → 故事2 → 故事3 → 人物评价

## 第四步：情绪促传播

### 4.1 核心任务

替读者说出他们想说但说不出口的话，表达他们内心的情绪。

### 4.2 情绪价值清单

文章要提供以下至少一种价值：
- 情感共鸣：让读者觉得"这就是我"
- 心理慰藉：缓解焦虑、提供温暖
- 身份认同：让读者感到被理解、被看见
- 情绪宣泄：替读者表达愤怒、委屈、不满

转发心理：读者转发是因为"作者帮我说了"、"这就是我的态度"。

### 4.3 金句制造

**1213式**：
- 结构：1个现象 + 2个对比 + 1个结论 + 3个字收尾
- 示例："我们这一代人，赶上了最好的时代，也赶上了最坏的时代，最终发现，活着就好。"

**否定法**：
- 结构：否定常识 + 给出新观点
- 示例："不是寒门难出贵子，而是这个时代不再需要贵子。"

**具象法**：
- 结构：抽象概念 + 具体场景
- 示例："成年人的崩溃，是从深夜加班后打不到车开始的。"

**类比法**：
- 结构：A像B一样 + 特征描述
- 示例："生活就像一杯温水，不烫手也不解渴，但你不得不端着。"

**反问法**：
- 结构：提出问题 + 不回答（留思考）
- 示例："如果努力真的有用，为什么最努力的往往不是混得最好的？"

**排比法**：
- 结构：三个并列 + 递进升华
- 示例："我们害怕失业，害怕生病，害怕孤独，最后发现，我们害怕的只是活着本身。"

**反转法**：
- 结构：铺垫期待 + 出人意料
- 示例："我曾经以为长大后就自由了，后来才发现，长大是失去所有自由的过程。"

## 低创作度内容规避指南（平台合规）

基于微信公众号官方标准，以下内容类型会被判定为低创作度，直接影响推荐机会。

### 一、四大低创作度类型

#### 1. 同质化内容

**❌ 违规表现：**
- 短期内频繁发布主题、标题、正文、封面高度相似的内容
- 套用固定模板批量生产（如"剧名+xx致死不知+而是"的固定格式）
- 账号整体重复率过高，缺乏信息增量

**✅ 正确做法：**
- 每篇文章有独特角度和信息增量
- 根据不同主题灵活调整结构和表达
- 保持内容多样性，避免让用户产生重复阅读困扰

**检查点：** 这篇内容与账号近期其他文章的区别度如何？读者能获得什么新信息？

---

#### 2. 抄袭搬运内容

**❌ 违规表现：**
- 直接截图搬运站内外内容，无自我观点
- 大篇幅复述他人内容，机械拼接、洗稿重组
- 轻加工后整体不含个人观点、分析等增量信息

**✅ 正确做法：**
- 借鉴基础上加入个人观点、分析和增量信息
- 原创内容为主，引用需注明并加入自己的思考
- 在借鉴的基础上进行二次创作和价值延伸

**检查点：** 这篇文章的核心观点是自己的吗？对他人内容的引用是否加入了新的思考？

---

#### 3. 低信息量内容

**❌ 违规表现：**
- 简单拼凑无意义信息，无阅读价值
- 图文无关或"看图说话"
- 引用存疑的数据/案例/言论
- 无依据的效果演示或夸大

**✅ 正确做法：**
- 确保每段内容都有实质价值
- 图片和文字相互配合、互为补充
- 使用可靠来源，数据需核实
- 实事求是，避免虚假宣传

**检查点：** 删掉这段话，文章会损失什么吗？每个段落都有存在的必要吗？

---

#### 4. 低质AIGC内容

**❌ 违规表现：**
- AI生成原文未作任何修改直接使用
- 以AIGC图片为主体，且未提供增量信息
- 借用AI批量制造低成本、低信息量内容

**✅ 正确做法：**
- AI用于参考资料、搜集灵感（辅助而非替代）
- 融入大量个人经验和思考
- 原创内容占比高，AI只是工具

**平台态度：** 不反对使用AI辅助创作，但反对直接生成、批量制造低成本内容。

**检查点：** 如果读者知道这是AI写的，会觉得被欺骗吗？AI内容占比是否过高？

---

### 二、写作前自检清单

**选题阶段：**
- [ ] 这个选题与近期内容是否有足够区分度？
- [ ] 能提供什么新的信息或视角？
- [ ] 核心观点是否原创？对他人观点的引用计划如何转化？

**素材阶段：**
- [ ] 数据来源是否可靠？是否经过核实？
- [ ] 案例是否真实？是否获得授权？
- [ ] AI辅助部分是否加入了个人思考？

**创作阶段：**
- [ ] 是否避免了固定模板套路？
- [ ] 每个段落是否有实质信息量？
- [ ] 图文是否相关且互补？

---

### 三、写作后质量评估

**四个核心维度评分（1-5分）：**

| 维度 | 5分标准 | 3分标准 | 1分标准 |
|-----|--------|--------|--------|
| **信息增量** | 提供全新视角或独家信息 | 有一定新信息但不够突出 | 纯复述已知信息 |
| **原创程度** | 核心观点原创，引用转化得当 | 部分原创，部分借鉴 | 大篇幅搬运或洗稿 |
| **内容密度** | 每段都有价值，无冗余 | 偶有废话但整体可接受 | 大量无意义拼凑 |
| **AI参与度** | AI仅作辅助，核心原创 | AI参与部分修改 | AI直接生成，未修改 |

**评分标准：**
- 16-20分：优质内容，可放心发布
- 12-15分：合格内容，建议优化后再发
- 低于12分：低创作度风险，需大幅修改

## 去AI味指南（Stop Slop）

### 禁止使用的表达

**清除开场废话**
- 不要用："Here's the thing"、"The truth is"、"It turns out"
- 不要用："Let me be clear"、"Can we talk about"
- 直接说内容，不要预告

**删除强调废话**
- 不要用："Full stop"、"Period"、"Let that sink in"
- 不要用："This matters because"、"Make no mistake"
- 不要用："Here's why that matters"

**禁用商业黑话**
- Navigate challenges → 处理、应对
- Lean into → 接受
- Game-changer → 重要的
- Deep dive → 分析
- Moving forward → 接下来

**删除所有副词**
- 不用：really, just, literally, genuinely, honestly
- 不用：simply, actually, deeply, truly, fundamentally
- 不用：inevitably, interestingly, importantly, crucially
- 不用：At its core, In today's, At the end of the day

**禁用套路结构**
- 不用："Not because X. Because Y."
- 不用："The answer isn't X. It's Y."
- 不用："It feels like X. It's actually Y."
- 不用："Not X. But Y."
- 直接说Y，不要否定X

**禁用表演式强调**
- 不用："Speed. Quality. Cost. You can only pick two. That's it."
- 不用："[Noun]. That's it. That's the [thing]."
- 用完整句子，相信内容本身

**禁用修辞套路**
- 不用："What if I told you..."
- 不用："Think about it"
- 不用："And that's okay"
- 直接陈述观点

**禁用虚假主语**
- 不用："The decision emerges"（决定不会自己出现，是人做的决定）
- 不用："The culture shifts"（文化不会自己变，是人改变了行为）
- 不用："The data tells us"（数据不说话，是人解读的）
- 明确指出是谁做了什么

### 写作检查清单

交付前检查：
- 有副词吗？删掉
- 有被动语态吗？找出主语，改主动
- 无生命的东西做了人类的动作？指明是谁
- 句子以Wh-词开头？重组
- 有"here's what/this/that"？删掉，直接说重点
- 有"not X, it's Y"？直接说Y
- 三个连续句子长度一样？打破一个
- 段落以短句结尾？换种结尾
- 有破折号？删掉，用逗号或句号
- 有模糊声明（"影响重大"）？指明具体影响

### 评分标准

1-10分评估：
- 直接性：陈述还是预告？
- 节奏：变化还是单调？
- 信任：尊重读者智商吗？
- 真实感：像人写的吗？
- 密度：有能删的吗？

低于35/50：重写

## 工作流程

1. **理解需求**：明确主题、目标平台、目标受众
2. **生成选题**：用选题框架提供3-5个方向，分析爆款潜力
3. **低创初筛**：检查选题是否存在同质化、抄袭风险（自我审查）
4. **制作标题**：为每个选题生成5个标题，用九大模板
5. **搭建结构**：选择最佳选题，用四大框架搭建文章结构
6. **撰写正文**：根据大纲撰写完整的文章正文（开头+正文各部分+结尾）
7. **植入情绪**：在正文中自然融入情绪起伏和金句
8. **去AI味检查**：对照Stop Slop指南清理AI腔
9. **低创评估**：按照四个维度评估内容质量
10. **输出方案**：按照输出格式给出完整方案+文章正文

**注意**：最终必须输出**完整的文章正文**（可直接发布或稍作修改即可发布），而不只是大纲。文章正文要符合目标平台风格，有传播力，无AI腔。

## 合规提醒

**平台明确反对：**
- 高度同质化、搬运抄袭、内容空洞、低质AIGC

**平台不反对：**
- 使用AI进行参考资料、搜集灵感等辅助创作

**核心原则：**
- 信息增量 > 套路模板
- 原创思考 > 机械搬运
- 辅助工具 > 直接生成
- 读者价值 > 发布数量

- 角度：用"假装上班"的场景切入

## 四、文章正文（完整版）

### 开头

早上9点，北京国贸的星巴克里坐满了人。

他们西装革履，面前放着笔记本电脑，时不时地接电话、回微信，看起来和普通的上班族没什么两样。但只有他们自己知道，这已经是他们"上班"的第30天了——没有工资、没有同事、更没有公司。

他们被裁员了，却不敢告诉家人。

于是，星巴克成了他们的"办公室"。一杯美式坐一天，假装一切还好。

成年人的体面，有时候就是这么脆弱。

### 正文

**第一部分：什么是"假装上班"？**

"假装上班"是这两年才出现的新现象。

35岁以上的职场人，尤其是互联网、金融行业的中层，成了裁员的重灾区。他们被优化、被毕业、被向社会输送人才。

但问题来了：怎么跟家里交代？

房贷要还，孩子要养，父母的医药费不能拖。告诉妻子失业了？告诉父母被裁了？不，他们选择沉默。

早上照常出门，晚上照常回家。只不过，目的地从写字楼变成了咖啡厅、图书馆、商场休息区。

星巴克是最受欢迎的"办公地点"。有WiFi、有电源、有空调，最重要的是——有体面。

点一杯最便宜的咖啡，就能坐一整天。穿得体面一点，没人知道你其实已经失业了。

**第二部分：三个"假装上班"的真实故事**

**故事一：张哥的30天**

张哥，38岁，某大厂前产品总监。年薪80万，两个月前被裁。

"不是绩效问题，是整个业务线砍了。"他说得轻描淡写，但眼神里全是迷茫。

他不敢告诉妻子。房贷每月2万，孩子国际学校一年20万。妻子是全职太太，如果知道他失业了，这个家就塌了。

于是，他每天早上8点准时出门，去家附近的星巴克"上班"。

投简历、面试、接猎头电话。有时候一天两个面试，有时候一个都没有。

"最难的是晚上回家，妻子问今天工作怎么样，我要编各种故事。"

30天后，他终于找到了新工作，薪资降了40%。但他觉得自己是幸运的，"至少不用继续演下去了"。

**故事二：李姐的双面人生**

李姐，42岁，前外企HR总监。被裁后，她在星巴克"上班"了整整三个月。

"HR是最知道市场行情的，我知道我这个年纪，这个薪资，很难了。"

她的策略是：白天在星巴克投简历、学习新技能，晚上回家照常做饭、陪孩子写作业。

"我不是不想告诉家人，我是不想让他们担心。"

三个月里，她面试了20多家公司，有的嫌她年纪大，有的嫌她薪资高，有的干脆告诉她"我们更想要男性"。

最后，她降薪50%去了一家创业公司。"至少我还在工作，至少我还有价值。"

**故事三：王叔的最后体面**

王叔，52岁，传统制造业中层。公司倒闭，他成了"失业中年人"。

和年轻人不同，他连"假装上班"的地方都很有限。不懂咖啡，不会用电脑，星巴克对他来说太陌生了。

他选择了商场的长椅。早上9点到，下午5点走。中午在商场美食城吃最便宜的套餐。

"我不敢去公园，怕遇到熟人。我不敢去图书馆，怕坐久了被赶。"

两个月后，他选择去送外卖。"至少不用演戏了，至少我是真的在赚钱。"

**第三部分：为什么会有"假装上班"？**

表面上看，这是失业者的尴尬。但深层看，这是整个社会的结构性问题。

第一，35岁危机真实存在。

互联网行业追求年轻，制造业追求低成本，金融行业追求高学历。35岁以上、薪资偏高的中层，成了"性价比最低"的群体。

第二，家庭责任太重。

这一代人，上有老下有小，房贷车贷压力大。失业不是一个人的事，是整个家庭的事。

第三，社会观念没有跟上。

我们还在用"有没有工作"来评价一个人的价值。失业=失败，待业=丢人。这种观念，让失业者不敢说实话。

**第四部分：谁来拯救"假装上班"的人？**

解决这个问题，不能只靠个人。

企业需要改变：35岁不是能力的终点，而是经验的起点。裁掉中年人，短期省了钱，长期丢了宝贵的经验。

社会需要包容：失业不是失败，只是暂时的停顿。我们需要更完善的失业保险、更友好的再就业培训。

家庭需要理解：如果你发现家人"不正常"，不要急着质问，先试着理解。那个每天在星巴克"上班"的人，可能比任何人都需要拥抱。

### 结尾

成年人的崩溃，往往是无声的。

他们不是没有眼泪，只是没有地方可以哭。他们不是没有困难，只是不敢说出口。

在星巴克里，每一杯美式都是一个未说出口的焦虑，每一个敲键盘的声音都是一次无声的叹息。

这个时代淘汰人的时候，连一声再见都不会说。但至少，在星巴克里，他们还能保持最后的体面。

如果你身边也有这样的人，请不要拆穿他。给他一点时间，给他一点空间，让他自己走出来。

毕竟，谁都有难的时候。
