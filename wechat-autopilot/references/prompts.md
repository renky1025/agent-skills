# WeChat AutoPilot 提示词模板

## 写作风格提示词

### 脱口秀风格（推荐）

```
模仿脱口秀演员的文风撰写文章：
- 语言犀利、通俗易懂
- 带点幽默和反讽
- 观点明确、立场鲜明
- 用口语化表达复杂概念
- 多用短句，有节奏感
- 适当使用网络流行语

文章结构：
1. 开头：抛出一个犀利观点或现象（Hook）
2. 展开：用具体例子支撑观点
3. 转折：给出反常识的看法
4. 结尾：金句总结 + 引发讨论的问题
```

### 新闻评论风格

```
以新闻评论风格撰写：
- 客观陈述事实
- 加入深度分析
- 引用多方观点
- 给出独到见解
- 数据支撑观点

文章结构：
1. 导语：概括新闻事件
2. 背景：交代来龙去脉
3. 分析：多维度解读
4. 观点：给出独立判断
5. 展望：预测发展趋势
```

### 科普教育风格

```
以科普教育风格撰写：
- 由浅入深讲解
- 多用类比和例子
- 分点列出要点
- 总结核心结论
- 避免专业术语堆砌

文章结构：
1. 引入：用一个场景或问题引入
2. 解释：通俗讲解核心概念
3. 举例：用生活化例子说明
4. 应用：说明实际用途
5. 总结：提炼关键知识点
```

### 商业分析风格

```
以商业分析风格撰写：
- 聚焦商业价值
- 分析市场机会
- 引用数据和案例
- 给出行动建议
- ROI 导向

文章结构：
1. 现象：描述市场/行业现象
2. 数据：用数据支撑分析
3. 机会：指出商业机会点
4. 案例：引用成功/失败案例
5. 建议：给出具体行动建议
```

## 文章优化提示词

### 标题优化

```
基于以下内容生成 5 个标题选项：
- 标题 1：使用数字，制造悬念
- 标题 2：蹭热点，引发好奇
- 标题 3：直击痛点，给出解决方案
- 标题 4：反常识观点，引发争议
- 标题 5：情感共鸣，引发共鸣

要求：
- 每个标题 20-30 字
- 包含关键词：{关键词}
- 适合在公众号传播
```

### 内容优化

```
优化以下文章，提升阅读体验：

1. 增加小标题，每 300 字一个
2. 关键数据加粗
3. 适当使用 emoji 分段
4. 增加过渡句
5. 结尾增加互动问题

原文：
{文章内容}
```

```
- 适合朋友圈分享

```

## 完整工作流提示词

### 完整流程提示词（给 AI 使用）

```
你是一个公众号自动化运营助手。请按照以下流程执行：

步骤 1：资讯收集
- 从 Google News RSS 获取关键词"{关键词}"的最新新闻
- 筛选出 5-10 条最相关的新闻
- 提取每条新闻的标题、摘要、链接

步骤 2：文章撰写
- 选择最热门的 3-5 条新闻作为素材
- 使用"{写作风格}"风格撰写文章
- 字数控制在 800-1200 字
- 包含：标题、导语、正文（3-4小节）、结尾

步骤 3：配图生成
- 为文章生成封面图描述
- 为文章生成 1-2 张正文配图描述
- 配图风格：{配图风格}

步骤 4：输出
- 输出完整的文章（HTML 格式）
- 输出配图描述（用于文生图 API）
- 输出推荐的标签（3-5 个）

请开始执行。
```

### 质量检查提示词

```
请检查以下文章是否符合公众号发布标准：

检查项：
1. 标题是否吸引人？（20-30字，有吸引力）
2. 内容是否原创？（非直接复制）
3. 是否有敏感词？（政治、色情、暴力等）
4. 事实是否准确？（数据、人名、公司名）
5. 排版是否清晰？（段落分明，有层次）
6. 是否有互动引导？（结尾有问题或邀请）

文章：
{文章内容}

请给出检查报告和改进建议。
```
