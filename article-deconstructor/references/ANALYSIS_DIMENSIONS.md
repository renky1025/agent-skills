# 分析维度详解

## 1. 核心观点 (Core Argument)

### 识别标准
- 文章反复强调的**单一中心思想**
- 去掉所有修饰后剩下的**本质信息**
- 能用一句话概括的**最大公约数**

### 分析要点
```
□ 核心观点是什么？
□ 观点的新颖程度（老生常谈/略有新意/颠覆认知）
□ 观点的反直觉程度（符合常识/略有冲突/完全颠覆）
□ 观点出现频率和位置（开头/中间/结尾）
□ 观点的包装方式（直白陈述/故事引出/数据支撑）
```

### 常见核心观点类型
| 类型 | 示例 | 效果 |
|-----|------|-----|
| 反常识型 | "越努力越幸运"是错的 | 挑战认知，引发好奇 |
| 解决方案型 | 3个方法解决XX问题 | 实用导向，承诺价值 |
| 情感共鸣型 | 成年人的崩溃都是静音模式 | 情绪共振，引发认同 |
| 身份认同型 | 真正聪明的人都... | 划分圈层，制造归属 |
| 趋势洞察型 | 2024年XX行业将发生巨变 | 制造紧迫感 |

## 2. 副观点/支撑论点 (Supporting Arguments)

### 分析框架
```
核心观点
    ├── 副观点1
    │   ├── 证据A
    │   └── 证据B
    ├── 副观点2
    │   ├── 案例C
    │   └── 数据D
    └── 副观点3
        └── 论证E
```

### 副观点关系类型
- **并列型**：多个角度平行支撑（第一...第二...第三...）
- **递进型**：层层深入（基础→进阶→高阶）
- **对比型**：正反对比（过去vs现在、错误vs正确）
- **因果型**：前因后果（因为...所以...）

### 分析清单
```
□ 列出所有副观点
□ 标注每个副观点的证据类型
□ 判断副观点之间的逻辑关系
□ 评估副观点对核心观点的支撑力度
```

## 3. 说服策略 (Persuasion Strategies)

### 说服策略分类

#### A. 数据说服
- 具体数字（3.5亿、87%）
- 统计结果（调查显示...）
- 研究引用（XX大学研究表明...）
- 时间数据（用时3年、每天10分钟）

#### B. 案例说服
- 个人故事（我当年...）
- 他人案例（我有个朋友...）
- 企业案例（XX公司...）
- 历史案例（当年...）

#### C. 权威说服
- 专家观点（XX教授说...）
- 名人背书（马斯克曾说...）
- 机构认证（世卫组织建议...）
- 数据平台（百度指数显示...）

#### D. 对比说服
- 前后对比（使用前vs使用后）
- 正反对比（成功的人vs失败的人）
- 极端对比（月入3千vs月入3万）
- 时间对比（十年前vs现在）

#### E. 情感说服
- 恐惧诉求（不这样做就会...）
- 希望诉求（只要这样做就能...）
- 愤怒诉求（凭什么...）
- 共鸣诉求（我们都经历过...）

#### F. 逻辑说服
- 因果链（因为A所以B所以C）
- 排除法（不是A不是B，只能是C）
- 三段论（大前提→小前提→结论）
- 归纳法（多个案例→一般规律）

## 4. 情绪触发点 (Emotional Triggers)

### 情绪类型对照表

| 情绪类型 | 触发词 | 身体反应 | 行为倾向 |
|---------|-------|---------|---------|
| 焦虑 | 担心、害怕、万一 | 心跳加速 | 寻求解决方案 |
| 恐惧 | 灾难、危机、毁灭 | 紧张、回避 | 立即行动/逃避 |
| 希望 | 可能、会好、转机 | 轻松、期待 | 尝试新方法 |
| 愤怒 | 不公、凭什么、太气人 | 血压上升 | 转发评论 |
| 共鸣 | 我也是、原来如此 | 被理解感 | 点赞收藏 |
| 好奇 | 为什么、秘密、真相 | 注意力集中 | 继续阅读 |
| 优越感 | 聪明、高级、只有1% | 自我感觉良好 | 分享彰显 |
| 归属感 | 我们、同类、懂的人 | 温暖感 | 评论互动 |

### 触发点标记模板
```
【触发点X】
位置：第X段第X句
原文："..."
触发情绪：[具体情绪]
触发机制：[痛点揭示/身份认同/利益承诺/恐惧唤醒]
情绪强度：★★★☆☆
停留时间：读者在此处会暂停思考
```

## 5. 金句提取 (Golden Sentences)

### 金句类型

#### 观点型
- 结构：直接陈述一个观点
- 示例："真正的人脉不是你认识多少人，而是多少人认识你。"
- 可复用句式：[真正的___不是___，而是___]

#### 共鸣型
- 结构：描述共同经历/感受
- 示例："成年人的崩溃都是静音模式。"
- 可复用句式：[成年人的___都是___]

#### Actionable型
- 结构：给出具体行动建议
- 示例："每天睡前问自己三个问题..."
- 可复用句式：[每天___问自己___]

#### 反转型
- 结构：先抑后扬或先扬后抑
- 示例："你以为的稳定，其实是最大的风险。"
- 可复用句式：[你以为的___，其实是___]

#### 金句型
- 结构：朗朗上口的短句
- 示例："选择比努力更重要。"
- 可复用句式：[___比___更重要]

### 金句提取模板
```
原文："..."
类型：[观点型/共鸣型/Actionable/反转型/金句型]
结构：[用___表示可变部分]
适用场景：[在什么类型文章中可用]
传播度：★★★★☆
```

### 情感曲线类型

#### A型曲线（跌宕起伏）

    ↑

    │   ╱ ╲      ╱ ╲

 低 │        ★★
    └──────────────────→ 文章进度
      开头   中间   结尾

适合：故事类、观点冲突类文章

#### B型曲线（稳步上升）
```

    ↑

    │                 ╱
    │              ╱
    │           ╱
 低 │★─────────
    └──────────────────→ 文章进度
      开头   中间   结尾
```
适合：干货分享、解决方案类文章

#### C型曲线（先抑后扬）
```

    ↑

    │      ★    ╱
    │     ╱ ╲  ╱

    └──────────────────→ 文章进度
      开头   中间   结尾
```
适合：痛点-解决方案型文章

    ↑

## 7. 情感层次 (Emotional Layers)

### 三层分析法

#### 表层（Surface Layer）
- 文字直接表达的意思
- 事实性信息
- 客观描述

#### 中层（Middle Layer）
- 暗示的情感
- 态度倾向
- 价值判断

#### 深层（Deep Layer）
- 触及的人性本质
- 普遍心理需求
- 文化/社会意义

### 分析示例
```
原文："月薪3万的人，下班后还在学习。"

表层：描述一个现象（月薪3万的人下班后学习）
中层：暗示你不成功是因为不够努力（焦虑唤醒）
深层：触碰"生存焦虑"和"向上流动"的渴望
```

## 9. 视角转化分析 (Perspective Shifts)

### 视角类型

#### 第一人称（我）
- 使用场景：分享个人经历、建立信任
- 效果：真实感、亲近感
- 风险：主观性强，说服力受限

#### 第二人称（你）
- 使用场景：直接对话、制造代入
- 效果：面对面感、针对性
- 风险：可能引起防御心理

#### 第三人称（他/她/他们）
- 使用场景：讲故事、客观陈述
- 效果：客观性、距离感
- 风险：情感连接较弱

### 视角切换分析模板
```
段落 │ 视角  │ 人称词 │ 使用目的
─────┼──────┼────────┼─────────
 1-3 │ 第一 │ 我     │ 建立信任
 4-6 │ 第二 │ 你     │ 制造代入
 7-10│ 第三 │ 他们   │ 客观论证
 结尾│ 第二 │ 你     │ 行动召唤
```

### 视角策略
- **我→你**：从分享转为对话（建立连接→引导思考）
- **你→他**：从对话转为观察（制造代入→提供证据）
- **他→你**：从观察转为对话（客观论证→行动召唤）

## 10. 语言风格特征 (Language Style)

### 句式偏好

#### 短句（<10字）
- 效果：节奏快、有力、易记忆
- 适合：金句、强调、开头结尾
- 示例："选择大于努力。"

#### 中句（10-20字）
- 效果：信息量适中，阅读舒适
- 适合：正文主体
- 示例："真正的人脉不是你认识多少人，而是多少人认识你。"

#### 长句（>20字）
- 效果：信息密度大，专业感
- 适合：复杂论证、数据引用
- 示例："根据XX大学2023年对1000名职场人的追踪研究显示，那些每天投入至少30分钟进行系统性学习的人，在三年后的薪资增长幅度平均比同龄人高出47%。"

### 词汇偏好

| 风格类型 | 特征词汇 | 适用场景 |
|---------|---------|---------|
| 专业风 | 术语、概念、理论 | 知识付费、行业分析 |
| 口语风 | 俗语、口语词、感叹词 | 情感故事、个人IP |
| 网络风 | 流行语、梗、表情包文字 | 年轻受众、热点话题 |
| 文艺风 | 诗意词汇、修辞、意象 | 深度好文、价值观输出 |
| 商务风 | 数据、指标、ROI | 商业分析、管理干货 |

### 修辞手法统计

```
比喻：X次（就像、好比、如同）
排比：X次（连续三个以上相似结构）
对偶：X次（前后呼应、结构对称）
反复：X次（关键词重复）
设问：X次（自问自答）
反问：X次（无需回答的反问）
```

### 节奏特征

```
整体节奏：[快/慢/起伏]

快节奏特征：
- 短句为主
- 段落简短
- 信息密集
- 适合：热点、时效性内容

慢节奏特征：
- 长句为主
- 段落较长
- 娓娓道来
- 适合：深度思考、情感故事

起伏节奏特征：
- 长短句交替
- 张弛有度
- 适合：故事叙述、观点论述
```
