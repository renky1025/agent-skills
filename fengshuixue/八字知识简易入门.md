# 八字知识简易入门

## 一、十天干在地支的根（通根）

### 核心概念
**"根"**指的是天干在地支中的支撑力量。就像树需要根才能生长，天干也需要在地支中有根才能强壮。

### 各天干在地支的根

| 五行 | 天干 | 比肩/禄（最强） | 劫财 | 其他根（较弱） |
|:---:|:---:|:---:|:---:|:---|
| **木** | 甲 | 寅 | 卯 | 未、辰、亥 |
| | 乙 | 卯 | 寅 | 未、辰、亥 |
| **火** | 丙 | 巳 | 午 | 戌、未、寅 |
| | 丁 | 午 | 巳 | 戌、未、寅 |
| **土** | 戊 | 巳 | 午 | 戌、未 |
| | 己 | 午 | 巳 | 戌、未 |
| **金** | 庚 | 申 | 酉 | 丑、戌 |
| | 辛 | 酉 | 申 | 丑、戌 |
| **水** | 壬 | 亥 | 子 | 辰、申、丑 |
| | 癸 | 子 | 亥 | 辰、申、丑 |

### 重要提示

1. **灰色字体**代表根气微弱，力量不强
2. **比肩（禄）做根最强**，也是自己的"禄位"

### 四墓库的特殊作用

- **未** = 木库：夏天生长的木，木可生火 → 可做**木和火**的根
- **戌** = 火库：内藏金 → 可做**火**的根，勉强做**金**的根
- **辰** = 水库：水能生木 → 可做**水和木**的根
- **丑** = 金库：金能生水 → 可做**金和水**的根

## 二、十神关系（以日主为中心）

### 核心口诀
> **生我者印，我生者食伤，克我者杀官，我克者财，同我者比劫**

### 十神速查表

| 关系 | 十神 | 同阴阳 | 不同阴阳 | 六亲 | 人物含义 | 其他含义 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---|
| **生我** | 印 | 偏印 | 正印 | 母亲 | 女性长辈 | 国家、房子、证书、单位、思想 |
| **同我** | 比劫 | 比肩 | 劫财 | 兄弟姐妹 | 同辈 | 竞争者、帮扶者、体力、四肢 |
| **我生** | 食伤 | 食神 | 伤官 | 子女 | 下属晚辈 | 口才、技术、自由、口福、快乐 |
| **我克** | 财 | 偏财 | 正财 | 父亲 | 男命老婆 | 钱财、机会、工作、荣誉、情绪 |
| **克我** | 杀官 | 七杀 | 正官 | 配偶 | 男命孩子 | 官方权威、上司、管理、压力、伤害 |

### 偏正区别
- **同性相生/相克** → 偏（偏印、偏财、七杀）
- **异性相生/相克** → 正（正印、正财、正官）

## 三、地支藏干（人元）

### 每个地支里藏什么？

| 地支 | 本气（最强） | 中气 | 余气（最弱） |
|:---:|:---:|:---:|:---:|
| 子 | 癸 | - | - |
| 丑 | 己 | 癸 | 辛 |
| 寅 | 甲 | 丙 | 戊 |
| 卯 | 乙 | - | - |
| 辰 | 戊 | 乙 | 癸 |
| 巳 | 丙 | 戊 | 庚 |
| 午 | 丁 | 己 | - |
| 未 | 己 | 丁 | 乙 |
| 申 | 庚 | 壬 | 戊 |
| 酉 | 辛 | - | - |
| 戌 | 戊 | 辛 | 丁 |
| 亥 | 壬 | 甲 | - |

### 记忆窍门

**四正位**（子、午、卯、酉）：只藏本气，纯净专一
- 子=癸、午=丁+己、卯=乙、酉=辛

**四长生**（寅、申、巳、亥）：藏三阳干或特定组合
- 寅=甲丙戊（木火土）、申=庚壬戊（金水土）
- 巳=丙戊庚（火土金）、亥=壬甲（水木）

**四墓库**（辰、戌、丑、未）：藏土+所墓之物
- 辰=戊乙癸（水库）、戌=戊辛丁（火库）
- 丑=己癸辛（金库）、未=己丁乙（木库）

## 四、十神类象（生活化解读）

### 1. 印星（正印、偏印）
- **关键词**：贵人、证书、房子、车子、母亲
- **像什么**：像"保护伞"，能给你资源和支持
- **生活场景**：学历证书、房产证、贵人相助、长辈关怀

### 2. 比劫（比肩、劫财）
- **关键词**：自己、朋友、同辈、身体、体力
- **像什么**：像"自己人"，同类相聚
- **生活场景**：兄弟姐妹、同学同事、合作伙伴、身体健康

### 3. 食伤（食神、伤官）
- **关键词**：孩子、学生、享受、说话、自由
- **像什么**：像"输出口"，表达和创造
- **生活场景**：演讲表达、艺术创作、吃喝玩乐、教导晚辈

### 4. 财星（正财、偏财）
- **关键词**：父亲、妻子、钱财、务实
- **像什么**：像"养命之源"，你追求的目标
- **生活场景**：工资收入、理财投资、男命的妻子、实际利益

### 5. 官杀（正官、七杀）
- **关键词**：丈夫、工作、压力、官方
- **像什么**：像"约束力"，规矩和压力
- **生活场景**：上司领导、工作职业、法律法规、女命的丈夫

## 总结

1. **通根**：天干在地支有支撑则强，无根则弱
2. **十神**：以日主为中心，看其他干支与我的关系
3. **藏干**：地支中藏有天干，影响命局力量
4. **类象**：十神对应生活中的各种人事物

> 💡 **学习建议**：先看懂日主是什么，再看其他干支与日主的关系，最后结合通根和藏干判断力量强弱。
